# 疯狂Java：突破程序员基本功16课 - 学习指南

## 课程概述

本书由李刚编著，是一本夯实Java基础、提升编程能力的经典书籍。通过16个专题，系统梳理Java核心技术点。

## 学习环境

- **JDK版本**: JDK 8
- **构建工具**: Maven 3.x
- **IDE**: IntelliJ IDEA / Eclipse

## 课程目录

### 第一阶段：面向对象基础（第1-4课）

| 课次 | 主题 | 核心知识点 |
|------|------|-----------|
| 第1课 | 类和对象 | 类定义、对象创建、构造器、封装 |
| 第2课 | 继承与多态 | 继承、super、final、向上转型、向下转型 |
| 第3课 | 字符串处理 | String、StringBuilder、StringBuffer、正则表达式 |
| 第4课 | 包装类 | 自动装箱拆箱、类型转换、缓存机制 |

### 第二阶段：核心API（第5-8课）

| 课次 | 主题 | 核心知识点 |
|------|------|-----------|
| 第5课 | 集合框架 | List、Set、Map、迭代器、排序 |
| 第6课 | 泛型 | 泛型类、泛型方法、泛型限定、通配符 |
| 第7课 | 异常处理 | 异常体系、try-catch-finally、自定义异常 |
| 第8课 | 反射 | Class获取、构造器/方法/属性反射、动态代理 |

### 第三阶段：高级特性（第9-12课）

| 课次 | 主题 | 核心知识点 |
|------|------|-----------|
| 第9课 | 枚举 | 枚举定义、枚举属性、方法、switch匹配 |
| 第10课 | I/O流 | 字节流、字符流、缓冲流、对象序列化 |
| 第11课 | 多线程 | Thread、Runnable、线程同步、线程通信 |
| 第12课 | 网络编程 | Socket、ServerSocket、TCP/UDP编程 |

### 第四阶段：数据库与新特性（第13-16课）

| 课次 | 主题 | 核心知识点 |
|------|------|-----------|
| 第13课 | JDBC | 连接管理、CRUD操作、事务、连接池 |
| 第14课 | Java 8新特性 | Lambda表达式、方法引用、Stream API |
| 第15课 | 函数式编程 | 函数式接口、闭包、函数组合 |
| 第16课 | 并发编程 | ExecutorService、Callable、Future、并发集合 |

## 学习建议

### 学习方法
1. **理论先行**: 先阅读章节内容，理解核心概念
2. **代码实践**: 每个知识点都要亲手编写代码验证
3. **举一反三**: 尝试对示例代码进行扩展和改造
4. **总结归纳**: 每课结束后整理笔记，形成知识体系

### 时间安排
- 基础阶段（1-4课）：建议1-2周
- 核心API阶段（5-8课）：建议2-3周
- 高级特性阶段（9-12课）：建议2-3周
- 新特性阶段（13-16课）：建议2-3周

### 代码运行
```bash
# 编译项目
mvn clean compile

# 运行测试
mvn test

# 单独运行某个类
mvn exec:java -Dexec.mainClass="com.learn.crazyjava.lesson01_class_object.Demo"
```
